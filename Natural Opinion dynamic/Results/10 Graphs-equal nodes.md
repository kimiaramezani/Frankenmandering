# Description
In this report, 10 different graphs with 240 nodes, each has 20 opinions are generated and each node is located in its own district. Two experiments have been done with these graphs: 
Experiment 1: Without normalization 
Experiment 2: With normalization
Graph size = 12 

DRF1 = f1  
DRF2 = DRF with social network
